For transit features, colors help distinguish lines!

Welcome to our Targeted Editing series where today you can hop on your favorite transit line, or make OpenStreetMap data divine! (Both take about the same amount of time.)

Read on for some basic uses for transit line colours, why I am using the British spelling, and where you can check to see if you can use your local knowlege to add colours where they are needed.

  1. How can transit line colours be used?

    1. Help with map rendering. Transit line colours can be used to make custom maps designed specifically for transit users.

    2. Help with wayfinding. A line colour on a map can help a traveler quickly understand generally where a route goes, without needing prior spatial knowledge of the origin or destination stops.

    1. Here is a map showing transit lines. To add a colour to a transit line, hover over one of the bright blue highlighted features to bring up an info bubble with links to editing tools that can be used to add features.

        See the wiki pages for colour for advantageous colour names and hex codes. You may also find features that are tagged "color" which isn't wrong, it's just not preferred. Consistency is really valuable in data, so when possible, choose the British spelling.

        But where do you add the colour tag? It should be added to a relation like the route_master. The route_master is a type of relation, and it contains all of the important information associated with the route. Alternatively, the colour tag can be added directly to individual ways.
